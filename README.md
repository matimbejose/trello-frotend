## Trello Frotend
Bem-vindo ao repositório frontend do Trello.



## Tecnologias usadas
- fortawesome:6.5.1
- axios: 1.6.8
- vee-validate: 4.12.6
- vue-router: 4.0.3
- vuelidate: 0.7.7
- vuex: 4.0.0




## Configuração do projeto
```
npm install
```

### Compila e recarrega automaticamente para desenvolvimento
```
npm run serve
```

### Compila e minifica para produção
```
npm run build
```

### Lint e corrige arquivos
```
npm run lint
```
